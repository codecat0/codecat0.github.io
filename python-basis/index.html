Python基础知识简单介绍

发表于 2019-08-08 | 分类于 python
字数统计: 2.5k | 阅读时长 ≈ 9

这篇文章适合于Python初学者来学习

基本语法元素

一，程序的格式框架

1.缩进

python语言采用严格的“缩进”来表示程序的逻辑

2.注释

python采用“#”表示一行注释的开始，多行注释需在每行开始都使用“#”

3.续行符

续行符由反斜杠（\）符号表示

等价于print("{}是{}的首都.format("北京","中国"))

二，语法元素的名称

1.变量

python语言中，变量可以随时命名，随时赋值，随时使用

2.命名

python采用大小写字母，数字，下划线和汉字等字符及其组合命名，但首字符不能是数字

3.关键字

and as assert break class continue def
del elif else except False finally for
form global if import in is lambda
None nonlocal not or pass raise return
True try while with yield async await

三.基本输入，输出函数

1.input()函数

input()函数从控制台获得用户的一行输入，无论用户输入什么内容，input()函数都以字符串类型返回结果

2.eval()函数

eval(a)函数将去掉字符串a最外侧的引号

3.print()函数

用于输出运算结果

基本数据类型

一，数字类型

1.整数类型

进制种类 引导符号 描述
十进制 无 123
二进制 0b或0B 由字符0和1组成，例如：0b1010，0B1010
八进制 0o或0O 由字符0-7组成，例如：0o1010，0O1010
十六进制 0x或0X 由字符0-9，a-f，A-F组成，例如：0x1010

2.浮点数

带有小数的值，例如：1010.0，1010.，1.01e3，1.01E3

3.复数类型

复数可以看作是二元有序实数对（a,b)，表示a+bj。注意：1，当b为1时，不能省略。2，a b都为浮点数

二，数字类型的运算

1.数值运算操作符

操作符 描述
x+y 和
x-y 差
x*y 积
x/y 商
x//y 整数商
x%y 余数
-x x的负值
+x x本身
x**y x的y次幂

2.数值运算函数

函数 描述
abs(x) x的绝对值
divmod(x,y) (x//y,x%y)
pow(x,y) x**y
round(x,d) 对x四舍五入，保留d位小数
max($x_1,x_2,…,x_n$) 求最大值
min($x_1,x_2,…,x_n$) 求最小值

abs(-30+40j)

结果为：50.0

三，字符串类型及格式化

字符串是字符的序列表示

1.索引

<字符串>[序号]
"abc"[1] = 'b'

2.切片

<字符串>[N:M]：切片获取字符串从N到M（不包含M）的子字符串
"abcdef"[1:3] = 'bc'

3.format()方法的格式控制

: 填充 对齐 宽度 ， .精度 类型
引导符号 用于填充的字符 <左对齐;>右对齐;^居中对齐 槽的设定输出宽度 数字的千位分隔符 浮点小数部分的精度或字符串的最大输出长度 整数类型：b,c,d,o,x；浮点数类型：e,f,%

四，字符串类型的操作

1.字符串操作符

操作符 描述
x+y 连接字符串x和y
x*n 复制n次字符串x
x in s 如果x是字符串s的字串，返回True，否则返回False

2.字符串处理函数

函数 描述
len(x) 返回字符串x的长度
str(x) 返回任意类型x的字符串形式
chr(x) 返回Unicode编码x对应的单字符
ord(x) 返回单字符x对应的Unicode编码
hex(x) 返回整数x对应的十六进制
oct(x) 返回整数x对应的八进制

3.字符串处理方法

方法 描述
str.slower() 返回字符串的副本，全部字符小写
str.upper() 返回字符串的副本，全部字符大写
str.split(sep) 返回一个列表，由str根据sep被分割的部分组成
str.count(sub) 返回sub字串出现的次数
str.replace(old,new) 返回字符串的副本，所有old字串被替换为new
str.center(width) 字符串居中函数
str.join(iter) 返回一个新字符串，将iter变量的每一个元素增加一个str字符串

五，类型判断与类型间转换

type(x)函数对变量x进行类型判断

函数 描述
int(x) 将x转换成整数
float(x) 将x转换成浮点数
str(x) 将x转换成字符串

程序的控制结构

一，程序的分支结构

1.单分支结构：if

2.二分支结构：if-else

3.多分支结构：if-elif-else

二，程序的循环结构

1.遍历循环：for

for 循环变量 in 遍历结构：

语句块

2.无限循环：while

while 条件：

语句块

3.循环控制：break和continue

break用来跳出最内层循环，然后执行循环后的代码

continue用来结束当前当次循环，即跳出循环体中下面尚未执行的语句，但不跳出当前循环

三，程序的异常处理

try:

语句块1

except：

语句块2

语句块1是正常执行的内容，当执行这个语句块时出错时，则执行语句块2

函数

一，函数的基本使用

def 函数名(函数参数):
函数体
return 返回值

二，函数的参数传递

1.可选参数的传递

注意：可选参数一般都放置非可选参数后面

2.参数名称传递

三，变量的作用域

变量分为局部变量和全局变量。局部变量仅在函数内部，且作用域也在函数内部。全局变量的作用域跨越多个函数。全局变量要在函数内部使用时，需提前用global关键字声明

注意：使用global关键字对全局变量声明时，该变量名与外部全局变量同名，且在函数内部对全局变量的操作会改变外部全局变量的值

组合数据类型

一，序列类型

操作符 描述
x in s 若x是s的元素，返回True
x not in s 若x不是s的元素，返回True
s + t 连接s和t
s*n 将序列s复制n次
s[i] 索引，返回序列第i个元素
s[i:j] 切片，返回序列s第i到j个元素的子序列（不含第j个元素）
s[i:j:k] 切片，返回序列s第i到j个元素以步长为k的子序列
len(s) 序列s的元素的个数
min(s) 序列s中最小元素
max(s) 序列s中最大元素
s.index(x) 序列s中第一次出现x的位置
s.count(x) 序列s中出现x的总次数

1.字符串【前面介绍了】

2.元组

元组类型在python语言中表示为tuple，一般以小括号和逗号组织，且一旦定义就不能修改

3.列表

列表的操作方法

方法 描述
ls.append(x) 在列表最后增加一个元素x
ls.insert(i,x) 在列表第i个位置增加元素x
ls.clear() 删除列表中所有元素
ls.pop(i) 将列表第i个位置的元素取出并从中删除
ls.remove(x) 将列表出现的第一个元素x删除
ls.reverse() 反转列表
ls.copy() 生成一个新列表，复制ls所有元素

二，字典类型

字典使用大括号或dict（）建立，每一个元素是一个键值对

操作方法 描述
d.keys() 返回所有键信息
d.values() 返回所有值信息
d.items() 返回所有键值对信息
d.get(key,default) 键存在就返回相应值，否则返回default
d.pop(key,default) 键存在就返回相应值，同时删除相应值，否则返回default
d.clear() 清空字典
d.popitem() 随机取出一对键值对，以元组形式返回，并将其从字典中删除

三，集合类型

集合类型与数学中的概念一致，用大括号表示，没有索引和位置的概念

集合类型操作符

操作符 描述
S-T 差集
S&T 交集
S^T 补集

集合类型主要用于去重，适合于任何组合数据类型

文件操作

1.文件的打开模式

打开模式含义 含义
‘r’ 只读模式，如果文件不存在，返回异常
‘w’ 覆盖写模式，文件不存在则创建，存在则完全覆盖原文件
‘x’ 创建写模式，文件不存在则创建，存在则返回异常
‘a’ 追加写模式，文件不存在则创建，存在就在原文件最后追加内容
‘b’ 二进制文件模式
‘t’ 文本文件模式，默认值
‘+’ 在原功能基础上增加同时读写功能

2.文件读取方法

方法 含义
f.read(size=-1) 读入整个文件的内容
f.readline(size=-1) 读入一行的内容
f.readlines(hint=-1) 读入所有行的内容
f.seek(offset) 改变当前文件操作的指针的位置。0：文件开头；2：文件结尾

3.文件写入方法

方法 含义
f.write(s) 向文件写入一个字符串
f.writelines(lines) 将一个元素为字符串的列表整体写入文件
